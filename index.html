Gemma-4 엔진 탑재

인터넷 연결 없이 기기 내부의 NPU와 CPU만으로 작동하는 초경량 자연어 추론 AI. 앱들을 분석해 직관적인 스마트 폴더 구조로 분류합니다.

AI 통합 파일 탐색

"어제 받은 영수증 찾아줘", "강아지 동영상 어딨어?" 등 자연어 텍스트만으로 파일 형태와 확장자를 스스로 추론해 검색합니다.

사용자 승인 기반 안전 제어

파일 관리나 임계 작업 시 시스템이 독단적으로 결정하지 않으며 명확한 이유를 화면에 띄우고 오직 승인 하에만 안전하게 실행됩니다.

하드웨어 제어 매직

"화면 밝기 조절해줘", "카카오톡 켜줘"와 같은 명령만으로 디바이스의 하드웨어 설정과 시스템 레벨 조작을 유연하게 제어합니다.

AI 스티커 공방

MediaPipe 연동 기반 이미지 지능형 탐색 및 누끼(배경 분리) 기능으로 제작한 커스텀 스티커를 화면 어디에나 배치 가능합니다.

강력한 엣지 패널 시스템

사이드 엣지 스와이프 전용 위젯 인터페이스와 GitHub Gist 웹 통신 패키지를 결합하여 실시간 버전을 동기화합니다.
